Demos¶

The hyperspy-demos git repository contains a number of helpful demonstrations on how to use HyperSpy. In order to get a feel for how HyperSpy works, the demos can be run locally on your machine or remotely without any installation necessary. For more information, follow the instructions in the repository’s README.
